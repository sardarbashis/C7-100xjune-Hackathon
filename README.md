# Concept Check

Track A submission for the 100x Engineers June Hackathon.

## Problem

Many learners can define technical concepts but struggle to derive them from first principles or explain why they exist.

## Solution

Concept Check is an AI-powered Socratic tutor that identifies gaps between memorized definitions and genuine conceptual understanding.

The system:

1. Asks the learner to explain a concept.
2. Detects weak reasoning or memorized explanations.
3. Generates targeted follow-up questions.
4. Evaluates whether understanding improves.
